# FPS_BOOST
Fps_Boost for qbcore. This script is helping for set the density of the local peds and cars.


## Installation:
```
    Drag and Drop
```

## Modification:
```
    Open the Client.lua change  the numbers from 0.0 to 1.0 
    
```
